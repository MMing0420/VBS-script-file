VBS is a scripting language based on Visual Basic. The full name of VBS is Microsoft Visual Basic Script Edition.  
VBS script files were invented by Microsoft to improve computer work efficiency, optimize computer processing procedures, automate operations, system management, web development, and more.  
However, it has both advantages and disadvantages; it can be exploited by trojan viruses to damage files, etc. Do not randomly open and run unfamiliar VBS files!  
Microsoft plans to retire the VBS script interpreter in the future.
VBS can be edited, for example, with VS or Notepad.
